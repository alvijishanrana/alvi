# 🌐 Personal Portfolio Website

This is a professional static portfolio website built to support academic, scholarship, and international applications. The portfolio highlights academic achievements, projects, teaching experience, certifications, leadership roles, and contact information.

---

## 🚀 Features

- Clean and professional design for global applications
- Structured sections:
  - Home
  - About
  - Academics
  - Projects
  - Experience
  - Awards & Achievements
  - Certifications
  - Leadership
  - Contact
- CV included as downloadable PDF (`cv.pdf`)
- Image placeholders for:
  - Profile photo
  - Project visuals
  - Certificates
- Fully static (HTML, CSS, JS) — easy to host anywhere

## 🖼️ Adding Your Images

Replace placeholder sections with your own images:

- Profile image → `assets/images/profile.jpg`
- Project images → inside project sections
- Certificates → upload and link in Certifications page

Make sure images are:
- High quality
- Professional (for scholarships)
- Properly named

---

## ✏️ Customization Guide

### 1. Update Content
Edit HTML files to:
- Add new projects
- Update achievements
- Modify contact info

### 2. Update CV
Replace:
```
cv.pdf
```
with your latest version.

### 3. Styling
Modify:
```
assets/css/style.css
```

## 🌍 Deployment Options

You can host this portfolio for free:

- GitHub Pages
- Netlify
- Vercel

### GitHub Pages (Quick Steps)
1. Push project to GitHub
2. Go to repository Settings → Pages
3. Select branch (main)
4. Your site will be live
